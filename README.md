# retention-analysis 📊
온라인 교육 서비스 운영 데이터 기반 재수강률 분석 프로젝트입니다.

---

## Background
온라인 교육 서비스 운영 중 재수강률이 주요 KPI로 관리되었으나, 데이터 기반 분석이 부족했습니다.

실제 운영 데이터를 기반으로 데이터를 직접 분석하여 재수강률 흐름 및 변동 요인을 분석했습니다.

---

## 분석 목표
- 월별 재수강률 변화 흐름 파악
- 학년별/상품별 재수강률 비교
- 이탈 유형 비교
- 담당자별 재수강률 차이 탐색
- 운영 KPI 흐름 분석 및 인사이트 도출

---

## 분석 환경
- Python, pandas, matplotlib
- Google Colab
- 분석 기간: 2024년 4월 ~ 2026년 4월

---

## 전처리
- 수학 정규 과목 필터링
- 개강 전 환불 데이터 제외
- 이상치 계약상태/상품명 제외
- 전환 → 연장완료로 통합
- 데이터 안정화 이전(~24-03) 및 미완료 월(26-05~) 제외
